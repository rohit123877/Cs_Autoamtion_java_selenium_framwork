# My Selenium Framework 
# This is a Selenium Automation Testing Framework developed using Java, Selenium WebDriver, TestNG, and Maven.

The framework includes:
	•	Base Class setup
	•	Executable Test Cases
	•	Extent Reports integration
	•	Screenshot capture on test failure
	•	TestNG Listener implementation

• The project follows a reusable and maintainable structure suitable for real-time automation projects.
• Data-Driven Testing: Integrated Apache POI to fetch test data directly from Excel sheets, allowing for extensive parameterization and test coverage.
• Advanced Logging: Utilized Log4j2 (Core & API) to maintain detailed execution logs, which is critical for debugging complex automation scripts.Modern Browser 	  • Management: Leveraged WebDriverManager to handle driver binaries automatically, ensuring compatibility across different environments.
• Rich Reporting: Implemented both Extent Reports (HTML) and Extent PDF Reports for high-quality, stakeholder-ready documentation of test results.Hybrid Framework  •Architecture: Combined the Page Object Model (POM) with TestNG for scalable, modular, and maintainable test execution.
