# My Selenium Framework 
# This is a Selenium Automation Testing Framework developed using Java, Selenium WebDriver, TestNG, and Maven.

The framework includes:
	•	Base Class setup
	•	Executable Test Cases
	•	Extent Reports integration
	•	Screenshot capture on test failure
	•	TestNG Listener implementation

The project follows a reusable and maintainable structure suitable for real-time automation projects.
